## What this is

This asset is a point release of the International Peace Information Service (IPIS) open-data table `cod_mines_curated_all_opendata_p_ipis`, covering artisanal mining sites in eastern Democratic Republic of the Congo. IPIS describes the table as field-collection data about mining sites, including site location, worker counts, minerals, traceability and qualification status, child and women labor indicators, state services, and the presence or interference of state and non-state armed actors.

The initial shared-datasets release uses the official IPIS CSV download that was last modified on 2026-05-26. The source CSV contains longitude, latitude, and an EWKB `geom` field. The canonical artifact writes the points as FGB geometry, keeps longitude and latitude as attributes, and drops the redundant EWKB source attribute.

## When to use it

- Use this as a citable point layer of IPIS open-data artisanal mining-site observations in eastern DRC.
- Use `vid` or `feature_id` for row-level lookup in this release.
- Use `pcode` to group records by IPIS mining-site identifier; do not assume it is row-unique.
- Use the FGB for analytical work and PMTiles for catalog or web-map display.
- Do not treat this as a complete census of all DRC artisanal mining sites, all DRC mining activity, or all mining activity outside the IPIS coverage area.

## Schema notes

The canonical FGB contains 7,163 point features in EPSG:4326. The release feature model adds `feature_id`, `geometry_hash`, and `properties_hash`. `feature_id` is copied from the source `vid` field because it is nonblank, unique, and URL-safe in this release. The source `pcode` field is documented by IPIS as the unique mining-site identifier, but it repeats across rows and is retained as a site grouping/search field instead of the row-level feature ID.

The source `geom` EWKB column is intentionally not published as an attribute. Canonical geometry is stored directly in the FGB feature geometry, and the original source row lineage is recorded in the metadata sidecar provenance.

No localized metadata sidecars were autogenerated for the first upload.
